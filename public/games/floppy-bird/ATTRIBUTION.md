# Floppy Bird asset attribution

This Darma playable uses local sprite and sound assets adapted from the open-source `jxmked/Flappybird` project package provided during QA.

- Source package: `Flappybird-xio.zip`
- Upstream project name: `jxmked/Flappybird`
- License file in source package: `BSD-3-Clause`
- Assets copied locally:
  - `atlas.png`
  - `sounds/wing.ogg`
  - `sounds/point.ogg`
  - `sounds/hit.ogg`
  - `sounds/die.ogg`
  - `sounds/swooshing.ogg`

The Darma game title and route were changed to `Floppy Bird` / `/games/floppy-bird`. The original `banner-flappybird` sprite is not rendered by the playable; the title is drawn as custom text to avoid showing the original title artwork.
