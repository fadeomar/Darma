# Buttons CSS Generator

Design CSS button styles visually — adjust colors, border-radius, padding, shadows, and hover effects — then copy the ready-to-use CSS.

## Privacy

`local-only` — all CSS generation runs in the browser. No data is sent anywhere.

## Files

| File | Role |
|---|---|
| `page.tsx` | Server component — `generateMetadata`, `ToolPage` |
| `ButtonsCssGeneratorClient.tsx` | `"use client"` — visual controls, live preview, copy output |
| `style.css` | Tool-scoped styles (preview sandbox resets) |
