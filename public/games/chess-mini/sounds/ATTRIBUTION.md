## How they are produced

These files are generated by [`scripts/games/generate-chess-sounds.mjs`](../../../../scripts/games/generate-chess-sounds.mjs)
(mono, 16-bit PCM, 22.05 kHz, normalized). Re-generate with:

```bash
node scripts/games/generate-chess-sounds.mjs
```

## Runtime behavior

At runtime the game prefers the Web Audio API synth in
[`chessSounds.ts`](../../../../src/features/games/playables/chess-mini/chessSounds.ts),
which produces the same chess-appropriate cues with normalized volume and no network
request. These `.wav` files are the graceful fallback for browsers without Web Audio
support. Audio never plays before a user gesture unlocks playback.
