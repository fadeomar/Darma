# Sky Hopper — Asset Attribution

Sky Hopper is an **original** Darma Games title inspired by the classic "tap to fly
through the gaps" arcade mechanic. It does **not** use any Flappy Bird branding, name,
sprite sheet, or audio — all artwork is drawn procedurally on an HTML canvas and all
sound is synthesized.

## Visuals

All graphics (sky, clouds, sun, obstacle pillars, ground, and the bird) are drawn at
runtime with the Canvas 2D API in
[`SkyHopperGame.tsx`](../../../src/features/games/playables/sky-hopper/SkyHopperGame.tsx).
No image files are used. **No third-party or copyrighted sprites.**

## Sounds

audio was downloaded, sampled, or hotlinked. They are released into the public domain

### How they are produced

Generated by
[`scripts/games/generate-sky-hopper-sounds.mjs`](../../../scripts/games/generate-sky-hopper-sounds.mjs)
(mono, 16-bit PCM, 22.05 kHz, normalized). Re-generate with:

```bash
node scripts/games/generate-sky-hopper-sounds.mjs
```

### Runtime behavior

At runtime the game prefers the Web Audio API synth in
[`skyHopperAudio.ts`](../../../src/features/games/playables/sky-hopper/skyHopperAudio.ts),
which produces the same cues with normalized volume and no network request. These `.wav`
files are the graceful fallback for browsers without Web Audio support. Audio never plays
before a user gesture unlocks playback, and the in-game mute toggle silences everything
(preference saved to localStorage).

> Note: because licensed third-party arcade audio could not be downloaded in the build
> environment, all cues are original CC0 assets. To swap in curated packs later (e.g.
> Kenney CC0, or Freesound CC0/CC-BY with attribution), drop the files here keeping the
> same filenames and update this table with source/author/license/URL.
