# Favicon & App Icon Generator

Complete client-only favicon and app icon studio for Darma.

## Features

- Generate from uploaded PNG/JPG/WebP/SVG, pasted SVG, text/initials, or emoji.
- Export Modern Web, Next.js App Router, PWA Complete, Legacy Full, or Complete Studio packs.
- Generate favicon.ico, PNG favicons, Apple touch icons, Android/PWA icons, maskable icons, monochrome icons, web manifests, snippets, README, and validation checklist.
- Preview browser tab, Google result, iOS home screen, Android/PWA, and maskable safe-area masks.
- Score icon readiness across source quality, browser favicon, Apple, PWA, maskable, manifest, and snippets.
- Validate existing files or ZIP packages locally.
- Quick setup presets for website launches, Next.js apps, installable PWAs, iOS shortcut packs, and complete brand kits.
- Install assistant with output-specific copy paths, deployment notes, and package size summary.
- Small-size legibility strip for checking 16px, 32px, 48px, Apple, and 512px outputs before download.

All generation runs in the browser. No server upload is used.
