# App Screenshot / Mockup Generator

Client-only Darma tool for turning raw app screenshots into polished mockups.

## Route

`/tools/app-screenshot-mockup-generator`

## Features

- Screenshot upload: PNG, JPG, WebP
- Optional background image upload
- Device frames: phone, tablet, laptop, desktop, browser, clean card
- Backgrounds: solid, gradient, mesh, image
- Text overlays: badge, title, subtitle, footer
- Frame controls: scale, rotation, radius, padding, shadow, fit mode, alignment
- Export packs: landing page, social promo, app store preview, documentation, complete kit
- Local ZIP export with README and snippets
- HTML, Next.js Image, and CSS snippets
- Readiness score and local package checker

## Privacy

All rendering and downloads run in the browser. No screenshot is uploaded.
